  <action>
    Create engine.py with RunEngine class. The core loop pattern:

    ```python
    class RunEngine:
        def __init__(self, experiment_dir: Path, config: Config, state: SessionState):
            self.experiment_dir = experiment_dir
            self.config = config
            self.state = state
            self.git = GitManager(experiment_dir)
            self.guardrails = ResourceGuardrails(config, experiment_dir)
            self.cost_tracker = CostTracker()
            self.deviation = DeviationHandler(config.direction)
            self.progress = LiveProgress(config, state)
            self._checkpoint_dir = experiment_dir / ".mlforge"
            self._journal_path = experiment_dir / "experiments.md"

        def run(self) -> None:
            with self.progress:
                try:
                    while not self.guardrails.should_stop(self.state):
                        save_checkpoint(self.state, self._checkpoint_dir)
                        result = self._run_one_experiment()
                        action = self._process_result(result)
                        self.state.experiment_count += 1
                        self.progress.update(self.state)
                        if action == "stop":
                            break
                finally:
                    save_checkpoint(self.state, self._checkpoint_dir)
                    self.git.close()
    ```

    **_run_one_experiment():** Build subprocess command list with all claude -p flags from config. Use subprocess.run with capture_output=True, text=True, timeout, cwd. Handle TimeoutExpired, non-zero returncode, JSONDecodeError. Return parsed JSON dict on success.

    **_process_result(result):** Extract total_cost_usd via result.get("total_cost_usd", 0.0) and record in CostTracker. Call DeviationHandler.handle(). Route:
    - "keep": self.git.commit_experiment(), update state.best_metric/best_commit, state.total_keeps += 1, state.consecutive_reverts = 0
    - "revert": self.git.revert_to_last_commit(), state.total_reverts += 1, state.consecutive_reverts += 1
    - "retry": call _run_one_experiment() again (with OOM hint)
    - "stop": return "stop"

    **_build_prompt():** Read experiments.md content. Construct prompt: "You are an ML research agent. Read CLAUDE.md for your protocol. Read experiments.md for history. Run train.py, evaluate results, and report the metric value. Metric: {config.metric} (direction: {config.direction}). Experiment #{state.experiment_count + 1}."

    **Signal handling:** Register SIGINT handler in run() that sets a _stop_requested flag. Check the flag in the loop alongside guardrails.

    Write tests using monkeypatch to mock subprocess.run. Create mock claude -p JSON responses for success, crash, timeout, and OOM scenarios. Verify the full loop behavior including checkpoint saves, keep/revert counts, and cost accumulation. Do NOT spawn actual claude -p processes in tests.
