```python
if args.swarm and args.resume:
    print("Error: --swarm and --resume cannot be used together", file=sys.stderr)
    return 1

    print("Warning: --n-agents has no effect without --swarm", file=sys.stderr)
```

3. Inside the try block, after scaffold + git init, add swarm branch BEFORE the existing RunEngine path:
```python
if args.swarm:
    from mlforge.swarm import SwarmManager
    manager = SwarmManager(config=config, experiment_dir=target_dir, n_agents=args.n_agents)
    manager.setup()
    try:
        results = manager.run()
        print(f"\nSwarm complete: {results['agents']} agents, "
              f"best={results['best_score']} (agent {results['best_agent']})")
        if results.get("verification"):
            v = results["verification"]
            status = "VERIFIED" if v.get("match") else "MISMATCH"
            print(f"Verification: {status}")
    finally:
        manager.teardown()
    return 0
```

The swarm branch must come AFTER scaffold_experiment() and git init (SwarmManager.setup needs an initialized git repo for worktrees), but BEFORE RunEngine instantiation.
