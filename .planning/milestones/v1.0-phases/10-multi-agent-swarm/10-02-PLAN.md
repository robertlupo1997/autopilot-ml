    Create src/automl/swarm.py following the SwarmManager skeleton from 10-RESEARCH.md exactly, with these specifics:

    SwarmManager class:
    - __init__(experiment_dir: Path, n_agents: int, task_type: str, metric: str, time_budget: int)
    - Cap n_agents at len(ALGORITHM_FAMILIES[task_type]) with a warning printed to stderr if exceeded
    - setup() -> list[list[dict]]: create dirs, create worktrees with run_tag = time.strftime("%Y%m%d-%H%M%S"), write config.json, return family assignments
    - run(assignments) -> None: register SIGINT handler, spawn agents, call _monitor_loop

    - _handle_sigint(sig, frame): set _shutdown=True, proc.terminate() all agents
    - _divide_families(families, n_agents): round-robin assignment
    - teardown(): remove_worktree for each agent (try/except to handle already-removed), then git worktree prune

    spawn_agent function (module-level):
    - Takes agent_id (int), workdir (Path), assigned_families (list[dict]), metric (str), time_budget (int), swarm_dir (Path)
    - Builds prompt string with agent ID, assigned family names, scoreboard path, metric, time budget
    - Returns subprocess.Popen with: claude -p <prompt> --allowedTools "Bash(*)" "Edit(*)" "Write(*)" "Read" "Glob" "Grep" --output-format json --max-turns 50
    - cwd=str(workdir), stdout=PIPE, stderr=PIPE
