   ```python
   from sklearn.datasets import load_iris
   import pandas as pd
   iris = load_iris()
   df = pd.DataFrame(iris.data, columns=[c.replace(" (cm)", "").replace(" ", "_") for c in iris.feature_names])
   df["species"] = iris.target
   df.to_csv("tests/fixtures/iris.csv", index=False)
   ```
   Run this script to produce the CSV. Verify 150 rows, 5 columns (4 features + species).

3. Create `scripts/` directory if it does not exist.

4. Create `scripts/run-baseline-test.sh` — a self-contained bash script the user runs OUTSIDE of Claude Code. The script must:

   a. Set strict mode: `set -euo pipefail`
   b. Define a PROJECT_ROOT variable (dirname of the script's parent)
   c. Define EXPERIMENT_DIR as a temp location or `$PROJECT_ROOT/experiment-iris`
   d. Copy `tests/fixtures/iris.csv` to a working directory
   e. Run `uv run automl iris.csv species accuracy --goal "Classify iris species (0=setosa, 1=versicolor, 2=virginica)"`
   f. `cd` into the scaffolded experiment directory
   g. Run `uv sync` to install dependencies
   h. Run `git init && git add . && git commit -m "initial scaffold"`
   i. Invoke the autonomous loop:
      ```bash
      claude -p "Follow the CLAUDE.md protocol exactly. NEVER STOP until max-turns is reached." \
        --max-turns 30 \
        --max-budget-usd 2.00 \
        --allowedTools "Bash Edit Read Write" \
        --output-format json \
        2>&1 | tee baseline-run-output.json
      ```
   j. After the run, capture post-run diagnostics:
      - `cat results.tsv` (if it exists)
      - `git log --oneline`
      - `git diff HEAD -- prepare.py` (should be empty — frozen file compliance)
      - Parse `baseline-run-output.json` for stop_reason, num_turns, total_cost_usd
   k. Print a summary of diagnostics to stdout
   l. Add comments explaining each step and the OBSERVATION CHECKLIST:
      - Did 3-5 drafts run? Were they different algorithm families?
      - Was best draft selected and checked out?
      - Did git commits happen on keep? Did reset happen on revert?
      - Did `grep "^metric_value:"` reliably return a float?
      - Is results.tsv being written? Correct format?
      - Was run.log captured? No context flooding?
      - Did agent touch prepare.py? (should be NO)
      - Did stagnation trigger strategy shift? (may not trigger in 30 turns)
      - If a crash happened, was it handled?
      - Were any tool uses blocked (permission denials)?
      - What was the stop_reason?

   m. Make the script executable: `chmod +x scripts/run-baseline-test.sh`

IMPORTANT: The script must print clear instructions at the start telling the user to run it from a terminal OUTSIDE of Claude Code (not from within a CC session), because `claude -p` cannot be launched inside another Claude Code session.
