  <name>Task 1: Create profiler module with DatasetProfile and auto-detection</name>
  <files>src/mlforge/profiler.py, tests/mlforge/test_profiler.py</files>
  <behavior>
    - Test: numeric target with >20 unique values detected as regression with metric="r2", direction="maximize"
    - Test: string/categorical target detected as classification with metric="accuracy" (binary) or "f1_weighted" (multi-class)
    - Test: numeric target with <=20 unique values detected as classification
    - Test: datetime64 columns detected as date columns
    - Test: string columns parseable as dates (ISO format) detected as date columns
    - Test: non-date string columns NOT detected as date columns
    - Test: profile reports correct n_rows, n_features, numeric_features, categorical_features, missing_pct
    - Test: empty DataFrame handled gracefully (no division by zero)
  </behavior>
  <action>
    Create src/mlforge/profiler.py with:

    1. DatasetProfile dataclass with fields: task (str), metric (str), direction (str), n_rows (int), n_features (int), numeric_features (list[str]), categorical_features (list[str]), date_columns (list[str]), target_stats (dict), missing_pct (float), leakage_warnings (list[str]).

    2. profile_dataset(df: pd.DataFrame, target_column: str) -> DatasetProfile function:
       - Task detection: if not pd.api.types.is_numeric_dtype(target) or target.nunique() <= 20 -> classification
       - Metric selection: binary classification -> "accuracy", multi-class -> "f1_weighted", regression -> "r2"
       - Direction: always "maximize" for these metrics
       - Feature type detection: separate numeric vs non-numeric columns (excluding target)
       - Date column detection via _detect_date_columns()
       - Missing data: count NaN cells / total cells as percentage
       - Target stats: for regression {type, mean, std}, for classification {type, n_classes, distribution}

    3. _detect_date_columns(df: pd.DataFrame, columns: list[str]) -> list[str]:
       - Check datetime64 dtype first
       - For object columns: sample head(20), pd.to_datetime(sample, errors="coerce"), accept if >80% parse
       - Wrap in try/except for robustness

    Write tests FIRST in tests/mlforge/test_profiler.py using synthetic DataFrames.
  </action>
  <verify>
    <automated>python3 -m pytest tests/mlforge/test_profiler.py -x -q</automated>
  </verify>
  <done>profile_dataset() correctly auto-detects task type (classification/regression), selects metric, detects date columns, and reports data characteristics. All 8+ test cases pass.</done>
</task>
