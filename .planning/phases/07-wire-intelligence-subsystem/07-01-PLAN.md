Modify RunEngine.run():

Add `_compute_baselines(self) -> dict | None` method:
- Guard: if `self.config.domain != "tabular"`, return None
- Guard: if prepare.py does not exist, return None
- Use `importlib.util.spec_from_file_location` to load prepare.py as a module

Modify RunEngine._process_result():
- After the "keep" action is determined but BEFORE committing, check baseline gate:
  ```python
  if self.state.baselines and metric_value is not None:
      if not passes_baseline_gate(metric_value, self.state.baselines, self.config.direction):
          # Downgrade to revert -- did not beat baselines
          self.git.revert_to_last_commit()
          self.state.total_reverts += 1
          self.state.consecutive_reverts += 1
          self._write_journal(exp_id, "revert", metric_value, None)
          self._record_result(exp_id, "revert", metric_value, None)
          return "revert"
  ```
- After the keep path (commit + state update), call `self._write_journal(exp_id, "keep", metric_value, commit_hash)`
- After the revert path (revert + state update), call `self._write_journal(exp_id, "revert", metric_value, None)`
- After the revert path, add stagnation check:
  ```python
  if check_stagnation(self.state, threshold=self.config.stagnation_threshold):
      untried = [f for f in ALGORITHM_FAMILIES if f not in self.state.tried_families]
      if untried:
          new_family = untried[0]
          trigger_stagnation_branch(self.git, self.state, new_family)
          self.state.tried_families.append(new_family)
  ```

Add `_write_journal(self, exp_id, status, metric_value, commit_hash, hypothesis="")` method:
- Compute diff: `get_last_diff(self.experiment_dir) if status == "keep" else None`
- Compute delta: `(metric_value - prev_best) if metric_value and prev_best else None` where prev_best is the best_metric BEFORE this experiment's update
- Create JournalEntry with all fields
- Call `append_journal_entry(self._journal_jsonl_path, entry)`
- Re-render: `entries = load_journal(self._journal_jsonl_path)` then `self._journal_path.write_text(render_journal_markdown(entries))`
