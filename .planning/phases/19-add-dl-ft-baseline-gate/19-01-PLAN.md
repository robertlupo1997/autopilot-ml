<objective>
Implement baseline computation and gate enforcement for Deep Learning and Fine-tuning domains so the dual-baseline gate applies to all domains, not just tabular.

Purpose: Closes MISS-01 from v1.0 audit. Currently `_compute_baselines()` returns `None` for non-tabular domains, meaning DL/FT experiments skip the baseline gate entirely and can keep sub-random-guess results.

Output: Two new baseline modules (deeplearning/baselines.py, finetuning/baselines.py), engine dispatch extension, and comprehensive tests.
</objective>

    - For perplexity: random_guess = float(vocab_size), untrained_model = 0.8 * float(vocab_size)
