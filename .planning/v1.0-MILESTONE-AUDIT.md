## Integration Check

### Verified (42 connections)
All core cross-phase wiring confirmed. Key paths:
- CLI -> profiler -> scaffold -> engine -> subprocess -> JSON parse
- Engine loop: checkpoint -> experiment -> process_result -> journal
- Baselines: compute -> gate -> stagnation -> branch
- Swarm: manager -> worktrees -> agents -> scoreboard -> verify
- Artifacts: template -> predictions.csv -> diagnostics; template -> joblib -> export

### Integration Gaps (3 items, non-critical)

**1. DL/FT artifact export path** (medium)
- `export_artifact()` hardcodes `best_model.joblib` filename
- DL template saves `best_model.pt`, FT saves adapter directory
- Effect: DL/FT export silently skipped (no crash)
- Affects: DL-04, FT-04, UX-03

**2. DL/FT swarm verification** (medium)
- `verify_best_result()` hardcodes `python train.py --eval-only`
- DL/FT templates have no `--eval-only` flag
- Effect: Verifier returns match=False for non-tabular (non-fatal)
- Affects: SWARM-04

**3. Leakage warning display** (low)
- `profiler.leakage_warnings` always empty (validate_no_leakage never called)
- Effect: CLI leakage warning block never triggered
- Affects: GUARD-06, UX-04
